Free and very simple currency converter plugin with 190+ currencies for your website from mconvert.net developers.

== Description ==

Exchange rates widget with 190+ world currencies and cryptocurrencies. 
All the data is used from opened sources, updates every hour and doesn’t overload the website.  
Widget displays exchange rate table for requested currency. Choose among 7 available languages: English, Deutsch, Українська, Русский, Français, Español, 中国. 
User friendly design is very intuitive and doesn’t require any complex operations. 

The Currency and cryptocurrency exchange rates widget powered by [mconvert.net](https://mconvert.net)

Code shortcuts make it easy to embed plugin for your website, blog etc. Of course it’s Free, doesn’t require any payment or personal information. Exchange rates widget provides you with up to date data from reliable sources.

All the data is coming from a 3rd party server (mconvert.net) so it doesn’t create an overload for your page and also doesn't store or save any data. Don’t hesitate and give it a chance.

== Installation ==

1. Install the "Currency/cryptocurrency exchange rates widget" plugin
2. Activate plugin in "Plugins page".
3. Open Widgets page in admin panel, add it to website
4. Configure plugin as you want.
